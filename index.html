Sistema Administrativo Pbro. "José de Jesús Espinoza" (Julio de 2023). Este sistema fue desarrollado utilizando PHP, MySQL y JavaScript y aplicando la metodología Scrum. El objetivo del proyecto fue solucionar deficiencias tecnológicas identificadas dentro de la institución. El problema que se presentaba era la poca organización en los procesos administrativos y académicos en la "Unidad Educativa 'Presbítero José de Jesús Espinoza'". Esto se evidenciaba en la carencia de un registro de asistencias para el personal, así como en un sistema de inscripción y calificación de estudiantes obsoleto y poco eficaz.
